# Validation Plan

## Main MVP Bet

Small business owners want a simple SEO scan that tells them the real issues hurting their website, and some of them will show interest in an automated fix after seeing those issues.

## MVP Experiment

Let users enter their website URL.

The product scans up to 5 pages and shows the top SEO issues in plain language.

After the report, show a clear call to action:

```text
Fix these issues for me
```

or

```text
Unlock one-click fixes
```

## Success Signal

The strongest early signal is a user clicking the fix call to action after seeing their SEO issues.

That means the user is not only curious about the audit, but interested in the solution.

## MVP Success Threshold

The MVP is successful enough to continue if it gets:

- 20 website scans
- 5 email submissions
- 2 pricing plan clicks

These are early validation numbers, not final product metrics.

## Funnel To Measure

Track these steps separately:

1. User completes website scan.
2. User clicks "Fix these issues for me".
3. User submits email.
4. User views pricing page.
5. User clicks a pricing plan or buy button.

A pricing click is stronger than an email submit because it shows buying intent.

## Pricing Page Rule

The pricing page is a fake-door demand test.

The MVP does not collect payment.

When a user clicks a plan, show a message like:

```text
This plan is not available yet. We will email you when one-click SEO fixes are ready.
```

The goal is to measure purchase intent before building the final product.

## What To Learn

- Do users understand the SEO issues?
- Do the issues feel important enough to fix?
- Do users click the fix call to action?
- Which issue types create the most interest?
- Are users willing to give an email or start payment to get fixes?

## Report Style To Validate

Issue detection should be rule-based.

AI should be used to explain issues and suggest fixes in simple language.

This tests whether users value a plain-language SEO coach, not only a technical audit list.

## What Not To Build Yet

- Real CMS integrations
- WordPress, Shopify, Wix, or Webflow one-click edits
- Full technical SEO automation
- Large site crawling
- Agency dashboards
